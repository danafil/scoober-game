# Scoober Game
Fun multiplayer game with numbers.

## Getting started in development mode

### Running the server: 

Open a terminal window

1. `cd server && npm install`
2. `npm start`

### Running the client:

Open a terminal window

1. `cd client && npm install`
2. `npm start`

Open http://localhost:3000/ in the browser and enjoy the game.

## Running with Docker

`docker-compose up -d`
Open http://localhost:3000/ in the browser and enjoy the game.
